2. 다음과 같은 계산기 코드가 있을 때, 상세하게 살을 덧붙이고, 예외처리와 모듈화를 진행하라.
```
if __name__ == "__main__" :
  while 1 :
    a = input()
    b = input()
    c = input()
    if b == "+" :
     print(a, "+", c, "=", a+c)
    elif b == "-" :
      print(a, "-", c, "=", a-c)
    elif b == "*" :
      print(a, "*", c, "=", a*c)
    elif b == "/" :
      print(a, "*", c, "=", a/c)
```
